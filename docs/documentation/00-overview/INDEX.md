# Overview

kol-chess is a web app built on a base Vite + React + Tailwind v4 stack (pnpm, react-router-dom, no design system). This `INDEX.md` *is* the overview — the folder reserves the `00-` namespace and needs no loose siblings.

This is where a reader starts. It states what the app does in a paragraph, then routes onward as sections are added, and to the sibling [[../../operations/INDEX|operations/]] for how it builds and ships.
